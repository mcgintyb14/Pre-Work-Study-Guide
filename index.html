HTML

  • HTML helps build the sturcture of a web page.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • In semantic HTML elements, the name of the element describes its content and the role it plays on a webpage.
  • When working with visible HTML elements, the content is contained between an opening tag and a closing tag.
  • An element refers to the entire element from opening tag to closing tag, whereas a tag refers only to what appears inside the angle brackets.

CSS

  • CSS helps determine the style of a web page
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A passing indicates how much space we want around the content inside an element.
  • Within a .css file, we can specify hundreds of different custom styles by specifying the element to be styled, followed by specifc commands that can change font color, text size, image size, etc.

Git

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • git status: checks what branch we are currently in.
  • git checkout -b branch name: creates a new branch and switches to it.
  • a repository holds code and other files related to your projects, such as a README or a license.

JavaScript

  • JavaScript performs the logic to communicate with the browser.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Using the Math function and Math.random() helps us select a random number in a function.
  • While conditional statements will execute automatically, a function will not execute until it is called by naming the function followed by "()" and a ";".
